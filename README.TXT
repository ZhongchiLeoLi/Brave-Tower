Inspired by MicRouge, we built a similar chess-styled, turn-based 
treasure hunting gamethat features:
- 100% Hand-drawn graphics frame by frame using Piskel
- Randomized terrain and monsters each level
- Unique movement patterns for each monster
- Traps and shields for strategic gameplay!

To play the game and read more about it:
